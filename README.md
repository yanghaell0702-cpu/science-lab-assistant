# 생명과학 탐구 설계 도우미
> **"고등학교 생명과학 자유탐구부터 소논문 실험 설계까지, AI가 가설과 변수 통제를 자동으로 점검하고 체계적인 실험 프로토콜을 완성해 주는 스마트 설계 도우미입니다."**
고등학교 생명과학 실험이나 자유탐구 과제를 준비할 때, 실험 조건을 잘못 설정하여 실패하는 문제를 해결해 주는 프로그램입니다. 사용자가 탐구하려는 질문을 입력하면 가설 수립부터 변수 설정, 대조군 검토, 실험 순서 정리까지 AI가 자동으로 확인하고 조언해 줍니다. 최종적으로 설계된 내용을 바탕으로 깔끔한 실험 보고서를 작성하여 종이로 인쇄하거나 컴퓨터에 저장할 수 있습니다.
이 프로젝트는 고등학교 생물 실험이나 과학 탐구 보고서를 준비할 때, 실험 장비와 통제 조건 설정이 미흡하여 실험에 실패하거나 결론 도출에 오류를 겪는 학생 및 연구원들을 돕기 위해 개발되었습니다. 
---
## 🌟 주요 핵심 기능
## 주요 기능
### 1. 💡 과학적 가설 생성 (Hypothesis Generator)
- 탐구하고자 하는 질문이나 의문을 입력하면, AI가 과학적 이론 배경을 기반으로 독립변수와 종속변수의 인과관계가 뚜렷한 **검증 가능한 가설**을 제안합니다.
1. **가설 도출 및 과학적 타당성 검토**
   - 탐구하고 싶은 질문을 입력하면 인과관계가 명확하고 실험으로 증명할 수 있는 가설을 추천합니다.
### 2. 🎛️ 변수 분석 및 오류 검증 (Variable Validator)
- 독립변수(조작변수), 종속변수(결과변수), 통제변수를 입력하면 AI가 변수 설계의 모순을 검출합니다.
- 특히 실험 결과의 신뢰도를 떨어뜨릴 수 있는 **누락된 통제변수(예: 온도, pH, 반응 시간 등)**를 정확히 짚어 보완 피드백을 제공합니다.
2. **독립변수, 종속변수, 통제변수 자동 검증**
   - 실험의 정확도를 위해 꼭 동일하게 유지해야 하지만 놓치기 쉬운 주변 환경 요인(온도, 농도 등)을 분석하여 통제변수로 추가하도록 지적해 줍니다.
### 3. ⚖️ 대조군 타당성 검토 (Control Group Review)
- 실험군과 대조군 설계를 진단합니다.
- 대조군이 비교 기준으로서 적합한지 평가하고, '실험의 신뢰도 극대화'를 위해 필수적인 **음성 대조군(Negative Control)** 설계 팁을 추천합니다.
3. **실험군과 대조군 타당성 검토**
   - 실험의 비교 기준이 되는 대조군과 변화를 주는 실험군이 알맞게 설계되었는지 분석하고 보완책을 제시합니다.
### 4. 📝 표준 실험 절차 자동 정리 (Protocol SOP Organizer)
- 머릿속에 있는 준비물과 대략적인 실험 순서를 적어 넣으면, 학술지 규격에 준하는 재현 가능하고 구체적인 **표준 실험 프로토콜(SOP)**과 **실험실 안전 가이드**로 문맥을 다듬어 정리해 줍니다.
4. **실험 절차 및 준비물 자동 정리**
   - 생각나는 대로 적은 거친 실험 순서와 준비물을 정리하여, 누구나 그대로 재현할 수 있는 명확한 단계별 실험 방법으로 다듬어 줍니다.
### 5. 🖨️ 보고서 생성 및 벡터 PDF 출력 (Report & Print)
- 앞선 단계에서 설정한 가설, 변수, 실험 설계 내용과 AI 피드백 요약본을 A4용지 규격의 **정식 과학 보고서 포맷**으로 통합합니다.
- 브라우저 인쇄(`window.print()`)를 연동하여 화면 깨짐 없이 원본 화질 그대로의 깔끔한 **PDF 보고서로 저장 및 인쇄**할 수 있습니다.
5. **실험 보고서 인쇄 및 저장**
   - 위 단계에서 작성한 모든 실험 설계 정보와 검토 의견을 모아 한 장의 보고서 서식으로 보여줍니다. 이를 곧바로 프린터로 인쇄하거나 파일로 저장할 수 있습니다.

## 프로그램 사용 방법
- **Frontend**: React (Vite), Vanilla CSS (Lab Tech 다크 테마 & Glassmorphism 디자인 적용)
- **AI Engine**: Gemini API (`gemini-2.5-flash` 모델 연동)

- **Print System**: CSS Print Stylesheet (`@media print` 최적화 레이아웃)

  > *(이와 함께 과학적 타당성 설명 및 고교 탐구 팁이 함께 제공됩니다.)*
## 사용 예시 (아밀레이스 효소 활성 실험)
---
실제 실험 설계를 진행할 때 입력할 수 있는 구체적인 입력 예시입니다.
### Step 2: 변수 통제
* **독립변수 (조작변수)**: `온도 (0°C, 37°C, 60°C)`
* **종속변수 (결과변수)**: `녹말 분해 속도 (아이오딘 반응 색깔 변화 시간)`
* **통제변수 (유지할 조건)**: `효소의 양, 녹말의 양`
* 입력 후 **[변수 설계 검증 및 검토]** 클릭
* **AI 출력 예시**:
  > **진단 결과 (보완 권장):** 설정된 변수 구조는 적절합니다. 하지만 정확한 실험을 위해 **pH 농도(완충 용액 사용)**와 각 시험관에 아밀레이스를 투입한 뒤 반응을 유도하기까지의 **온도별 예열 시간**을 통제변수로 반드시 추가해 주세요.
### [프로젝트 생성]
- **탐구 주제**: `온도가 침 아밀레이스의 녹말 분해 효소 활성에 미치는 영향`
- **탐구 개요**: `온도 변화에 따른 아밀레이스의 녹말 분해 속도를 비교한다.`
---
### Step 3: 대조군 검토
* **실험군**: `온도를 비정상적 수준(0°C, 60°C)으로 조정한 시험관들`
* **대조군**: `인체 온도와 유사한 37°C 최적 온도 조건을 유지한 시험관`
* 입력 후 **[대조군 타당성 검증]** 클릭
* **AI 출력 예시**:
  > **진단 결과 (타당함):** 비교 분석 설계가 우수합니다. 단, '아밀레이스 효소 자체'에 의해 녹말이 분해된 것인지 증명하기 위해 효소를 넣지 않은 `[녹말 용액 + 증류수]` 시험관을 추가 음성 대조군(Negative Control)으로 두어 비교할 것을 권장합니다.
### [단계별 설계 및 입력 데이터]
---
#### 1단계: 가설 설정
* **탐구 질문**: `온도가 침 아밀레이스의 녹말 분해 효소 활성에 미치는 영향`
* **하는 일**: 질문 입력 칸에 위의 내용을 붙여넣고 버튼을 누릅니다. AI가 과학적으로 적합한 가설을 제안해 줍니다.
### Step 4: 실험 절차 작성
* **준비물**: `녹말 용액, 아밀레이스 용액, 아이오딘-아이오딘화 칼륨 용액, 시험관, 얼음, 온수, 타이머`
#### 2단계: 변수 통제
* **독립변수 (변화를 줄 조건)**: `온도 (0도, 37도, 60도)`
* **종속변수 (결과로 나타날 현상)**: `녹말 분해에 걸린 시간`
* **통제변수 (똑같이 유지할 조건)**: `효소의 양, 녹말의 양`
* **하는 일**: 세 항목을 입력하고 검증 버튼을 누릅니다. AI가 온도 조건 외에 반드시 동일하게 유지해야 하는 추가 조건(액체 산성도인 pH 수치 등)이 무엇인지 알려줍니다.
#### 3단계: 대조군 검토
* **실험군**: `온도를 매우 낮추거나(0도) 끓인(60도) 시험관`
* **대조군**: `인체 온도와 비슷한 37도를 동일하게 유지한 시험관`
* **하는 일**: 입력창에 넣고 검증을 클릭합니다. AI가 대조군 설정의 잘된 점과, 효소를 아예 넣지 않은 대조 실험관을 추가해 보라는 등의 팁을 줍니다.
#### 4단계: 실험 절차 작성
* **준비물**: `녹말 용액, 아밀레이스 용액, 아이오딘 용액, 시험관, 얼음, 따뜻한 물, 타이머`
* **실험 방법 구상**:
  ```text
  1. 시험관에 녹말 용액을 각각 3ml씩 넣는다.
  2. 각각 얼음물, 37도 물, 60도 물에 두고 온도를 맞춘다.
  3. 아밀레이스를 섞어주고 1분마다 녹말 반응 시간을 측정한다.
  1. 시험관 3개에 녹말 용액을 3ml씩 채운다.
  2. 각각 얼음물, 37도 물, 60도 끓는 물에 둔다.
  3. 아밀레이스를 섞어준 뒤 1분 간격으로 아이오딘 용액을 한 방울씩 떨어뜨려 색이 변하는 시간을 잰다.
  ```
* 입력 후 **[표준 실험 절차 정리]** 클릭
* **AI 출력 예시**:
  > 학술용 규격에 맞는 **Materials(준비물 목록)** 정리 및 세부 행동 요령(SOP), 가열 장치 및 생체 시료 다루기 관련 **Safety Tips(안전 가이드)**가 완벽하게 다듬어져 리스트로 나열됩니다.
* **하는 일**: 구상 내용을 입력하고 정리 버튼을 누릅니다. AI가 안전 수칙을 포함하여 완벽한 시간과 용량이 적힌 정식 프로토콜로 문장을 다듬어 정리해 줍니다.
---
#### 5단계: 보고서 확인 및 인쇄
* **하는 일**: 최종 화면으로 넘어가 깔끔하게 모인 실험 설계 보고서를 확인합니다. '인쇄' 버튼을 눌러 컴퓨터에 저장하거나 종이로 출력합니다.
### Step 5: 보고서 생성 및 PDF 다운로드
* 작성된 모든 이력과 AI 피드백을 담은 **A4 형식 과학 보고서**가 생성됩니다.
* **[보고서 인쇄 / PDF 저장]**을 누르면 브라우저의 기본 인쇄 미리보기가 팝업됩니다.
* 인쇄 설정에서 **PDF로 저장**을 선택하면 보고서가 컴퓨터에 즉시 다운로드됩니다.
